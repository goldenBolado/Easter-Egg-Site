aside -> representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais. Elas muitas vezes contem explicações laterais, como a definição de um glossário; conteúdo vagamente relacionado, como avisos; a biografia do autor; ou, em aplicações web, informações de perfil ou links de blogs relacionados.

header -> É através dos cabeçalhos (headers) que informações adicionais podem ser enviadas na requisição e na resposta. Um header é composto por um nome, um signal de : e um valor.

CSS -> CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web.

mockup -> Mockup é o que os designers comumente utilizam para apresentar uma ideia, que se fosse explicada em palavras, por exemplo, não apresentaria, com clareza, o produto ou resultado final. Isso vale tanto entre participantes de uma equipe de projeto, quanto na comunicação com o cliente final.

flexbox -> Mais conhecido por devs web, já que o flex vem lá do CSS, o conceito do Flexbox é o de organizar elementos dentro de containers de forma flexível. Essa flexibilidade se caracteriza pela capacidade de alterar a largura e a altura dos itens conforme a necessidade para que o layout se adeque a qualquer tela.

gallery -> Uma galeria onde as imagens ficarão dispostas em seu site.

footer -> representa um rodapé para o seu sectioning content (conteúdo de seção) mais próximo ou sectioning root elemento.

HTML -> HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo.



article -> Artigo HTML5 é um elemento semântico HTML5, semelhante a <seção> e <cabeçalho>. É mais comumente usado para conter informações que podem ser distribuídas independentemente do restante do site ou aplicativo em que aparecem.

grid -> A grid no CSS oferece um sistema de layout baseado em grade, com espaços e colunas, deixando mais fácil o web-design sem ter a necessidade de usar floats e posicionamentos.

news -> Notícia em inglês
